# Morning Light - Daily Devotion App

Morning Light is a serene and user-friendly web application designed to provide daily devotional content. With a focus on a clean interface and readability, it offers a peaceful space for users to engage with spiritual readings. The app is fully bilingual, supporting both English and Tamil.

## Features

*   **Bilingual Content**: Seamlessly switch between English (EN) and Tamil (TA) with a modern toggle switch. The entire interface and devotion content adapts to your selected language.
*   **Daily Devotions**: Access a new devotion for each day. Easily navigate between days using previous/next buttons or select a specific date with the calendar picker.
*   **Powerful Search**:
    *   Quickly find devotions using the intuitive search bar centrally located on the homepage.
    *   Search results are displayed in a clear, card-style format, showing the result count and a relevant content snippet.
    *   Searched keywords are highlighted in yellow for easy identification within the results.
    *   Clicking a search result navigates directly to the devotion page, smoothly scrolls to the keyword, and temporarily highlights it for context.
*   **Customizable Reading Experience**:
    *   **Light & Dark Modes**: Choose between a light theme for daytime reading or a soothing dark theme for the evening with a simple toggle.
    *   **Adjustable Font Size**: Increase or decrease the text size to suit your reading comfort.
*   **Modern & Responsive UI**:
    *   A clean, minimalist design that works beautifully on desktops, tablets, and mobile devices.
    *   Consistent and aesthetically pleasing UI elements, including toggle switches and icons.
    *   Informative messages with a "Not Available" icon for dates where content is not available.
*   **Rich Content Formatting**: Devotional text correctly displays markdown formatting like **bold** and *italics*. Biblical verses are presented in a distinct, elegant style with the Georgia font and a mild background to stand out.
*   **External Resources**: Quick links to supplementary materials like YouTube videos or PDF documents related to the devotion.
